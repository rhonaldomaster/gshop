Stack para uso en produccion

- Almacenamiento de archivos: [Cloudflare R2](https://www.cloudflare.com/es-es/developer-platform/products/r2/)
- Hosting admin web y seller: [Vercel](https://vercel.com/)
- Hosting backend: [Railway](https://railway.com/)
- Hosting base de datos: [Supabase](https://supabase.com/)
- Streams: [AWS IVS](https://aws.amazon.com/es/ivs/)
